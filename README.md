# Big Six Virtual Office Dashboard

## Tech Stack
- Next.js 14 (App Router)
- React + TypeScript
- Tailwind CSS
- Supabase (real-time data)
- Vercel (deployment)

## Features
- Virtual office layout with agent avatars
- Real-time status updates
- Click-to-interact with any agent
- Activity feed
- Business switcher (DNA, Defy, Blacktop)

## Agent Avatars
- Oliver: Herbal/wellness themed
- Trey: Transportation/trucking themed
- Miles: Tech/platform themed
- Social Media: Creative/content themed
- Lead Scraping: Data/research themed
- Email: Communication themed
- CRM: Organization/pipeline themed
